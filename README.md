# 项目简介

## 核心功能
- 基于 YOLOv8 的骑行头盔检测系统
- 支持图片、视频及摄像头进行检测
- 支持选择模型
- 界面实时显示目标位置、目标总数、置信度等信息
- 支持批量检测并直接查看所有检测结果
- 支持检测结果保存
- 支持选择单张图片或图片文件夹进行识别
- 支持选择视频文件进行识别
- 支持调用本地摄像头进行识别
- 支持自定义置信度、IOU阈值
- 支持选择显示标签和原图
- 支持选择检测模型

## 适用场景
- 骑行安全检测
- 交通监控管理
- 安防系统应用
- 智能识别系统开发

## 技术特点
- 使用 Pycharm+Anaconda 或 VSCode+Anaconda 开发环境
- 基于 Python 3.9、OpenCV-Python、PyQt5、Ultralytics、Torch1.9 等技术栈
- 包含完整源码、124张骑行头盔数据集（4类：带头盔、不戴头盔、骑行、号码牌）
- 训练好的模型及训练结果
- 提供项目运行教程和UI界面源文件
- 支持跨平台部署运行
